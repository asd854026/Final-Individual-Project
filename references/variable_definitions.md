# Variable Definitions

## PerceptionOfWeight

`PerceptionOfWeight` is the independent variable for this project. It describes how students perceive their own weight.

| Code | Meaning |
|---:|---|
| 1 | Very underweight |
| 2 | Slightly underweight |
| 3 | About the right weight |
| 4 | Slightly overweight |
| 5 | Very overweight |

In the analysis notebooks, this variable is also recoded into `WeightPerception`, a readable category label.

## BMIPCT

`BMIPCT` is the dependent variable for this project. It is a quantitative BMI percentile variable.

## Analysis Roles

| Variable | Role | Type | Use in Project |
|---|---|---|---|
| `PerceptionOfWeight` | Independent variable | Categorical | Defines the five comparison groups |
| `BMIPCT` | Dependent variable | Quantitative | Outcome used to compare group means |

## Research Question Connection

These variables are used to answer the question: Do students' perceptions of their weight reflect their actual BMI percentile?
