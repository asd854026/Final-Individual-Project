# One-Slide Summary Content

The actual one-page summary deliverable is the PowerPoint file:

- `report/one_slide_summary.pptx`
- `outputs/summary/one_slide_summary.pptx`

This slide is designed as a single infographic slide for exhibition. It includes the research question, group definition, response variable, method, key result, conclusion, and one clear figure. It avoids long paragraphs and is intended to be understandable in about 30-60 seconds.
